Hiding history

 Ok this is kinda complicated but i wanna brag about it so im gonna. In older versions of this website every time you would change pages when searching, a new tab in history would pop up, and if someone clicked on it, it would bring them to what you were looking at. Now the url stays the same when you change pages, meaning only 1 tab shows up in history, and if someone got suspicious and clicked it, theyd see a blank page intentionally named "untitled". While this isnt perfect it's absolutely the best history hiding method currently. Something worth mentioning is that you should bookmark this page (g3marchi.github.io) not any is.gd links
